# angular-elements-ivy
Angular Elements Ivy Demo Example


## Getting Started
* Clone repository
  ```
  git clone https://github.com/aelbore/angular-elements-ivy.git
  npm install
  ```
* Build
  ```
  npm run build
  ```
* Start the Application
  ```
  npm start
  browse: http://localhost:3000
  ```
